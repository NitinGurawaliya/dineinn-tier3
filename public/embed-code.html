DineInn Merchant Banner - Embed Code

This code can be embedded on your Zayka Store website to help restaurant owners discover DineInn.

Option 1: Iframe Embed (Recommended)

Add this code to your website's HTML:

<iframe src="https://yourdomain.com/merchant-banner.html" width="100%" height="80" frameborder="0" scrolling="no" style="border: none; overflow: hidden;"></iframe>

Option 2: Direct HTML Embed

Or copy this HTML directly into your page:

<div style="background: linear-gradient(135deg, #2563eb 0%, #1d4ed8 100%); color: white; padding: 1rem; box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1);"> <div style="max-width: 1200px; margin: 0 auto; display: flex; align-items: center; justify-content: space-between; flex-wrap: wrap; gap: 1rem;"> <div style="display: flex; align-items: center; gap: 0.75rem;"> <svg style="width: 1.5rem; height: 1.5rem; fill: currentColor;" viewBox="0 0 24 24"> <path d="M12 2L2 7l10 5 10-5-10-5zM2 17l10 5 10-5M2 12l10 5 10-5"/> </svg> <div> <h3 style="font-weight: 600; margin: 0; font-size: 1rem;">Are you a Restaurant Owner?</h3> <p style="margin: 0; font-size: 0.875rem; color: #bfdbfe;">Create digital menus and manage your restaurant with DineInn</p> </div> </div> <div style="display: flex; align-items: center; gap: 0.75rem;"> <a href="https://yourdomain.com/?type=merchant" target="_blank" rel="noopener noreferrer" style="background: white; color: #2563eb; padding: 0.5rem 1rem; border-radius: 0.5rem; text-decoration: none; font-weight: 500; font-size: 0.875rem;">Get Started</a> </div> </div> </div>
Important: Replace yourdomain.com with your actual DineInn domain name in both code examples above.

Features

  • Responsive Design: Works on all device sizes
  • Auto-hide: Banner automatically disappears after 30 seconds
  • User Preference: Remembers if user closed the banner
  • Clean Integration: Minimal impact on your website's design

Customization

You can customize the banner by modifying the CSS styles in the HTML code. The banner uses a blue gradient background but can be changed to match your website's color scheme.
